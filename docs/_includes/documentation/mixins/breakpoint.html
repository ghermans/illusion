### @mixin breakpoint Sets a temporary global `$inside-breakpoint` variable to be used inside other mixins. #### Arguments | Name | Type | Default | Description | | ---- | ---- | ------- | ----------- | | `width` | `string` | - | ... | | `width--max` | `string` | false | ... | #### SCSS {% highlight css %} .foo { @include breakpoint($beta) { width: auto; } } {% endhighlight %} #### CSS {% highlight css %} @media screen and (min-width: 560px) { .foo { width: auto; } } {% endhighlight %}
