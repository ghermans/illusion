### @mixin flexbox Only apply styling when flexbox is either available (`@include flexbox`) or not (`@include no-flexbox`). Uses modernizr. O #### SCSS {% highlight css %} .foo { @include flexbox($beta, $charlie) { display: flex; } } .foo { @include no-flexbox { display: table; } } {% endhighlight %} #### SCSS {% highlight css %} @media (min-width: 560px) and (min-height: 768px) { .flexbox .foo, .flexboxlegacy .foo { display: flex; } } .no-flexbox .foo, .no-flexboxlegacy .foo { display: table; } {% endhighlight %}
